# vue_drinks_app

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### Install Tailwindcss 
```
yarn add -D tailwindcss@latest postcss@latest autoprefixer@latest
```

### Install daisyui 
```
yarn add daisyui
```

### Install heroicons
```
yarn add @heroicons/vue
```
